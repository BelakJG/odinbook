# Odinbook

[Project Link](https://www.theodinproject.com/lessons/ruby-on-rails-odin-book)

This project is to build a clone of a social media site.
It will feature user authentication, using either [OmniAuth](https://github.com/omniauth/omniauth) through the original social media site or using a gem like [Devise](https://github.com/heartcombo/devise).

The purpose is to take a higher level set of requirements and turn them into a fully functional site.
